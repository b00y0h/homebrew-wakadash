## Installation

```bash
# Tap the repository
brew tap b00y0h/wakadash

# Install wakadash
brew install wakadash
```

Or install directly in one command:

```bash
brew install b00y0h/wakadash/wakadash
```

## Security Note

The wakadash binary is currently unsigned. Homebrew automatically removes the macOS quarantine attribute during installation to prevent "damaged application" warnings.

You can verify the source code and checksums:
- Source code: https://github.com/b00y0h/wakadash
- Release checksums: https://github.com/b00y0h/wakadash/releases
